# Termux-Basic-Setup
## How To Run:
1. `pkg install git`
2. `git clone https://github.com/ziomek-z-ulicy/Termux-Basic-Setup.git`
3. `cd Termux-Basic-Setup`
4. `chmod +x Start.sh`
5. `./Start.sh` 
6. Done!
